パノラマ画像をドラッグ＆ドロップ

設定JSON（.json）をドロップするとプロジェクトを復元できます

または

JPEG / PNG / WebP ・ 推奨サイズ：4096×2048（2:1 正距円筒図法）・ 複数ファイル同時選択可

📂

設定JSONからプロジェクトを開く

JSON + 画像ファイル一式があれば別環境でもプロジェクトを完全復元できます。JSONと画像を一緒にドロップすると自動マッチングします

操作方法

  • 🖱️ ドラッグで視点を移動（タッチ端末はスワイプ）
  • 🔍 スクロール／ピンチでズーム
  • ⌨️ キーボードショートカット（ビューワー表示中）
← →前のシーン / 次のシーン（通常表示時）
R視点リセット
A自動回転 ON / OFF
F全画面切り替え
M現在シーンを左右反転
C分割比較 ON / OFF
Sスライダー比較 ON / OFF
V分割比較中：左右 ↔ 上下切替
L比較中：視点同期 ON / OFF
Esc通常表示に戻る

※ D5 Render の360°出力は「正距円筒図法」を選択してください。
