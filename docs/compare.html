⊞ 比較モード

2つのシーン（例：施工前と施工後、提案Aと提案B）を並べて見比べることができます。「分割比較」と「スライダー比較」の2種類があります。

比較を始める

  • ツールバーの「⊞ 分割比較」または「◫ スライダー比較」をクリックします（キー C / S）。
  • 比較中はA・Bそれぞれのサムネイル付きボタンをクリックすると、シーン一覧から比較対象を選び直せます。
  • 「⇄ A/B入替」でAとBを入れ替えられます。

表示の切り替え

比較中でも、A・Bの組み合わせを維持したまま表示方法だけを切り替えられます。

  • 分割 ⇔ スライダー： ツールバーの切替ボタンで表示方式のみが変わり、比較対象（A/B）は変わりません。
  • 左右 ⇔ 上下： 分割比較中のレイアウトボタン、またはキー V で切り替えられます。
仕様： 「A vs B → 分割 → スライダー → 上下 → 左右 → 分割 → …」と何度切り替えても、常に同じ A・B の組み合わせのまま表示方法だけが変化します（v2.9.1で修正）。

視点の同期

「🔗 視点同期」を ON にすると、A・B両方の視点が連動して動きます（キー L）。OFFにすると個別に操作できます。

比較セットの保存

気に入った比較の組み合わせは「💾 セット保存」で名前を付けて保存できます。保存される内容は次の通りです。

項目内容
sceneAId / sceneBId比較対象の2シーン
mode分割 / スライダー
layout左右 / 上下（分割時）
sliderPositionスライダーの位置
syncViews視点同期の ON/OFF

サイドバーの「比較セット」一覧からクリックすると、保存した状態を完全に復元して呼び出せます。

VRモードについて： 比較VR（分割比較・スライダー比較のVR表示）は現バージョンでは対象外です。分割比較・スライダー比較中は「🥽 VRモード」ボタンが無効化されます。通常表示（単体シーン）に戻すと有効になります。詳しくは「VRモード」を参照してください。
