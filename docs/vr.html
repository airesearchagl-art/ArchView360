🥽 VRモード（単体VR）

Meta QuestなどのWebXR対応ブラウザでは、現在表示中の360°パノラマシーンを1つだけ、HMDの向きで見回せる没入表示（VRモード）に切り替えられます。

推奨環境

推奨： Meta Quest Browser（Quest単体のブラウザ）で、HTTPS上にホストされたArchView360を開いてVRモードを使用してください。WebXRはQuest単体ブラウザでは対応しやすく、最も安定して動作します。

PC編集 → Quest確認の推奨フロー

シーンの作成・編集はPCで行い、VRでの確認はQuestで行うのが推奨の運用です。

  1. PCでArchView360を開く
  2. 画像・平面図・マーカー・比較セットを設定する
  3. 設定（プロジェクトJSON）を書き出す
  4. Quest BrowserでArchView360を開く
  5. 書き出したJSONと画像一式を読み込む
  6. VRモードで確認する

Quest Link / USB接続 / PC画面拡張について

Meta Quest LinkやUSB接続でPC画面をHMDに拡張表示する運用は、非推奨（検証扱い）です。

  • 通常の画面拡張は、ArchView360にとってのVR表示（WebXRセッション）ではありません。PC画面がHMD内に映っているだけです。
  • VRモードを使うにはWebXRの immersive-vr セッションが必要です。画面拡張だけではこのセッションは開始されません。
  • PCブラウザ＋Quest Link経由では、HMDがWebXRデバイスとして安定して認識されない場合があり、環境依存が大きいため推奨しません。

対応環境

環境VRボタン
Meta Quest Browser（Quest 2 / 3 / Pro等）有効（WebXR対応）
その他のWebXR（immersive-vr）対応ブラウザ有効
通常のPCブラウザ（Chrome/Edge/Safari等、HMD未接続）無効（自動でdisabled）
HTTP（非HTTPS）環境無効・VR開始不可
仕様： 起動時に navigator.xr.isSessionSupported('immersive-vr') でWebXR対応を判定します。非対応ブラウザではツールバーの「🥽 VRモード」ボタンが無効化され、ボタンのツールチップに「このブラウザはWebXR VRに対応していません」と表示されます。

使い方

  1. パノラマ画像、またはJSONプロジェクトを読み込みます。
  2. 通常表示でVRで見たいシーンを選択します。
  3. 上部ツールバーの「🥽 VRモード」ボタンをクリックします（WebXR対応ブラウザのみ有効）。
  4. VR表示に切り替わります。HMDの向きで360°空間を見回せます。
  5. VRを終了すると、自動的に通常表示（マウス操作）に戻ります。

Meta Questでの操作手順

  1. Quest BrowserでArchView360を開く
  2. パノラマ画像またはJSONプロジェクトを読み込む
  3. 通常表示で見たいシーンを選択する
  4. 上部ツールバーの「🥽 VRモード」を押す
  5. VR表示に入る
  6. HMDの向きで見回す
  7. 終了すると通常表示へ戻る

通常表示に戻る方法

HMDまたはブラウザ側のVR終了操作（Questの場合は本体ボタン操作でのセッション終了等）でVRセッションを終了すると、ArchView360は自動的に通常表示に戻ります。VR中に自動回転は一時停止され、終了後は元の設定に復帰します。

対象外の機能（今回のバージョンでは非対応）

  • 比較VR（分割比較・スライダー比較のVR表示）
  • FloorMap VR（平面図のVR表示）
  • コメントVR
  • mp4パノラマ動画のVR表示
  • INSP / INSV などInsta360ネイティブ形式のVR表示
仕様： 分割比較中・スライダー比較中はVRボタンが無効化され、ツールチップに「VRモードは通常表示のみ対応です」と表示されます。通常表示に戻ると再度有効になります。VR表示中かどうかの状態はプロジェクトJSONには保存されません（一時的な閲覧状態のため）。

トラブルシュート

症状原因・対処
VRボタンが押せない（disabled）ブラウザがWebXRに対応していない、またはHTTPS環境でない可能性があります。Meta Quest BrowserなどのWebXR対応ブラウザ＋HTTPS環境でお試しください。
「このブラウザはVR表示に対応していません」と表示されるnavigator.xr が存在しないブラウザです。WebXR対応ブラウザでアクセスしてください。
「VRセッションを開始できませんでした」と表示されるHMDが接続されていない、ユーザー操作なしでの起動、またはブラウザ側でVRセッションの許可が得られなかった可能性があります。VRボタンを直接クリックして再試行してください。
比較モード中にVRボタンが無効仕様です。通常表示（単体シーン表示）に戻すとVRボタンが有効になります。
VR中に画面が固まる／映像が動かないVRセッションを終了し、ページを再読み込みしてから再度お試しください。
