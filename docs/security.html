🔒 セキュリティポリシー

ArchView360はサーバーを使わない、ブラウザ完結型のツールです。読み込んだデータがどこにも送信されない設計になっています。

データの取り扱い

データ外部送信保存先
パノラマ画像なしブラウザのメモリ内のみ
平面図（FloorMap）画像なしブラウザのメモリ内のみ
プロジェクトJSONなしユーザーが明示的に書き出した場合のみローカルファイルとして保存
比較セットなしブラウザのlocalStorage内のみ
ローカル完結型ツール： ArchView360にはサーバー側のバックエンドが存在しません。画像・平面図・JSONはすべてブラウザ内（クライアントサイド）で処理され、インターネット上のどこにもアップロードされません。

なぜ安全なのか

  • ファイル読み込みはブラウザの File API（ローカル処理）のみを使用しています。
  • ネットワーク通信を行うのは、アプリ本体（HTML/CSS/JS）とビューワーライブラリ（Three.js）の配信時のみです。
  • 読み込んだ画像・JSONを送信するAPIエンドポイントは存在しません。
  • プロジェクトの共有は、JSON＋画像ファイルをユーザー自身が任意の方法（メール・共有フォルダ等）で送付することで行います。

注意事項

ブラウザのタブを閉じる、またはページを再読み込みすると、読み込んだ画像やシーン構成は破棄されます（永続化されません）。重要なプロジェクトは「JSON書き出し」で随時保存してください。
