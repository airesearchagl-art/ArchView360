⌨️ キーボードショートカット

ビューワー画面表示中に使えるキーボードショートカットの一覧です。

表示・ナビゲーション

キー動作
← →前のシーン / 次のシーンへ切替（通常表示時）
R視点リセット
A自動回転 ON / OFF
F全画面切り替え
M現在シーンを左右反転
Esc通常表示に戻る（比較モード/メニューを閉じる）

比較モード

キー動作
C分割比較 ON / OFF
Sスライダー比較 ON / OFF
V分割比較中：左右 ↔ 上下のレイアウト切替
L比較中：視点同期 ON / OFF

マウス・タッチ操作

操作動作
ドラッグ / スワイプ視点を回転
ホイール / ピンチズーム
シーンのダブルクリックシーン名をその場で編集
FloorMapピンの右クリック番号変更・名称変更・削除などのメニューを表示
ツールバー「🥽 VRモード」WebXR対応ブラウザで現在のシーンをVR表示（詳細）
ツールバー「👁 Observer」VR体験者の位置・視線方向をFloorMapで確認するパネルを開く（詳細）

VR中のQuestコントローラー操作（v2.13）

VRセッション中はキーボードショートカットの代わりに、Questコントローラーで操作します。詳細は VRモード を参照してください。

操作動作
トリガー（左右どちらのコントローラーでも可）次のシーンへ（← →キーと同じナビゲーション順）
スクイーズ（グリップ）短押しVR HUDの表示/非表示を切り替え
スクイーズ（グリップ）長押し（約0.6秒以上）前のシーンへ
注意（v2.13.1）： A/B/X/Yボタンやハンド判定には依存せず、トリガー・スクイーズのみで動作します。入力が反応しない場合は、VR HUD下部の [debug] select:N squeeze:N 表示でボタン入力自体が届いているか確認してください。
ヒント： ArchView360本体右上の「❔ クイックヘルプ」ボタンからも、この一覧の抜粋をすぐに確認できます。
