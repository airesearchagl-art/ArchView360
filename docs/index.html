ArchView360 ユーザーマニュアル

D5 Renderなどで書き出した360°パノラマ画像を、ブラウザだけで確認・比較・管理できる軽量ビューワーです。インストールや会員登録は不要で、画像をドラッグ＆ドロップするだけで使い始められます。

このマニュアルの内容

🚀
はじめに

画像の読み込み方、基本操作、対応形式を解説します。
🗂️
シーン管理

追加・削除・並び替え・グループ化・画像差し替えの方法。
🗺️
FloorMap

平面図にシーンをマーカーで配置し、ナビゲーションする方法。
⊞
比較モード

2つのシーンを分割・スライダーで比較し、セットとして保存する方法。
📂
プロジェクトJSON

プロジェクト設定を書き出し・読み込みして共有・復元する方法。
⌨️
キーボードショートカット

よく使う操作をキーボードだけで行うための一覧。
🔒
セキュリティポリシー

画像・JSONは外部送信されません。すべてブラウザ内で処理されます。
ヒント： ArchView360 本体の画面右上にある「❔ クイックヘルプ」ボタンでも、よく使う操作を簡単に確認できます。
ArchView360 User Manual — README も参照してください。
