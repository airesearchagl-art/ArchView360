🚀 はじめに

ArchView360はブラウザだけで動く360°パノラマビューワーです。サーバーへのアップロードは行われず、選んだ画像はそのままブラウザ内（メモリ）で表示されます。

対応している画像

項目内容
形式JPEG / PNG / WebP
推奨投影方式正距円筒図法（Equirectangular）、推奨サイズ 4096×2048（2:1）
ファイルサイズ1ファイルあたり 100MB 以下
複数選択可（一度に複数のシーンを追加できます）

D5 Render など多くのレンダラーには「360°パノラマ（正距円筒図法）」での書き出しオプションがあります。書き出し時にこの形式を選んでください。

画像を読み込む

  1. トップ画面の点線エリアに画像ファイルをドラッグ＆ドロップする。
  2. またはエリア内の「ファイルを選択」ボタンからファイルダイアログで選ぶ。
  3. 読み込みが完了すると自動的にビューワー画面に切り替わります。

基本操作

  • 視点の操作： マウスでドラッグ（タッチ端末はスワイプ）して見回せます。
  • ズーム： マウスホイール、またはタッチ端末ではピンチ操作。
  • シーン切替： 左サイドバーのシーンをクリック、またはキーボードの ← →。
  • 視点リセット： ツールバーの「リセット」、またはキー R。
  • 左右反転： 鏡面合成などで反転している画像を直す場合に使用（キー M）。
ヒント： 設定JSON（プロジェクトファイル）と画像をまとめてドロップすると、シーン構成・FloorMap・マーカー・グループなどをそのまま復元できます。詳しくは「プロジェクトJSON」を参照してください。
